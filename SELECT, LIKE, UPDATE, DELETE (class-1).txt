Clause in DBMS:-
1>Condition shows all details of those who have more than 70 marks?
SELECT * FROM STUDENT WHERE Marks >70;
2>Condition shows the name who has more than 70 marks?
SELECT Name FROM STUDENT WHERE Marks >70;

And operator use:-
1>Condition shows all details of those who have more than 70 marks but less than 90?
SELECT * FROM STUDENT WHERE Marks >70 AND Marks <90;
2>Condition shows the name of those who have more than 70 marks but less than 90?
SELECT Name FROM STUDENT WHERE Marks >70 AND Marks <90;
3>Condition shows all details of those who have more than or equal to 70 marks but less than or equal 90?
SELECT * FROM STUDENT WHERE Marks >=70 AND Marks <=90;
4>Condition shows the name of those who have more than or equal to 70 marks but less than or equal 90?
SELECT Name FROM STUDENT WHERE Marks >=70 AND Marks <=90;

Between operator use:-
1>Condition shows all details of those who have more than or equal to 70 marks but less than or equal 90?
SELECT * FROM STUDENT WHERE Marks BETWEEN 70 AND 90;
2>Condition shows the name of those who have more than or equal to 70 marks but less than or equal 90?
SELECT Name FROM STUDENT WHERE Marks BETWEEN 70 AND 90;

Like Clause use:-
1>Start with 'A' Name. 
SELECT Name FROM STUDENT WHERE Name LIKE 'A%';
2>End with 'x' Name.
SELECT Name FROM STUDENT WHERE Name LIKE '%x';
3>Anywhere with 'k' Name.
SELECT Name FROM STUDENT WHERE Name LIKE '%k%';
4>Show only name middle 'i' but not first 'N'
SELECT Name FROM STUDENT WHERE Name LIKE '%I%' AND Name NOT LIKE 'N%';
5>Show only the name who 3rd character 'i'
SELECT Name FROM STUDENT WHERE Name LIKE '%i__';
6>Show only the name who 2nd character 'n'
SELECT Name FROM STUDENT WHERE Name LIKE '_n%';
